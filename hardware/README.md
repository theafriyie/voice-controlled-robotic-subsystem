# Hardware

This folder contains hardware information for the Voice-Controlled Robotic Subsystem.

## Hardware Components

- Arduino Nano 33 BLE Sense
- built-in microphone
- 28BYJ-48 stepper motor
- ULN2003 motor driver
- jumper wires
- USB connection for programming and power

## Hardware Role

The hardware setup allows the system to capture spoken commands and convert them into physical motor movement.
